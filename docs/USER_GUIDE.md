# Руководство пользователя TempNoteManager

TempNoteManager помогает быстро разобрать файлы, открытые в Notepad++: временные заметки, постоянные файлы и файлы, которые нужно разложить по хранилищам-категориям.

Главная идея: слева находится рабочий список файлов, справа открывается выбранный файл, а категории помогают физически разложить файлы по директориям.

## Перед началом

Рекомендуется закрыть Notepad++ перед операциями, которые меняют `session.xml` или физически перемещают файлы.

Это особенно важно для действий:

- `Порядок`
- `Сохранить как...`
- `Сделать временным`
- `Удалить`
- перемещение файла в категорию
- возврат файла из категории в общий список
- переименование файла

Если Notepad++ открыт, он может позже перезаписать `session.xml` своими текущими данными.

## Первый запуск

1. Запустите `TempNoteManager.exe`.
2. Проверьте путь в поле `session`.
3. Если путь не найден автоматически, нажмите `...` рядом с `session` и выберите `session.xml` Notepad++.
4. При необходимости выберите папку `категории`.
5. Нажмите `Сохранить`.
6. Нажмите `Обновить`, чтобы перечитать список файлов.

Обычно `session.xml` находится здесь:

```text
%APPDATA%\Notepad++\session.xml
```

По умолчанию корневая папка категорий создается в документах:

```text
%USERPROFILE%\Documents\TempNoteManager Categories
```

## Главный экран

В верхней части окна находятся настройки:

- `session` - путь к `session.xml` Notepad++.
- `AI` - включает или выключает AI-функции.
- `корзина` - выбирает способ удаления.
- `категории` - корневая папка для хранилищ-категорий.

Основная область состоит из двух частей:

- слева список файлов;
- справа панель полного просмотра выбранного файла.

Если закрыть правую панель кнопкой `x`, список снова занимает больше места.

## Режимы списка

Список можно смотреть в двух режимах:

- `Таблица`
- `Карточки`

В таблице доступны колонки:

- `Имя`
- `Тип`
- `Создан`
- `Изменен`
- `Размер`
- `Категория`
- `AI тэги`
- `Путь`

Для длинных путей используйте горизонтальную прокрутку таблицы.

В карточках удобнее быстро просматривать краткий текст, summary, тэги и принадлежность к категории.

## Поиск и фильтр

Используйте строку поиска над списком, чтобы отфильтровать файлы по имени, пути, категории, тэгам или summary.

Флажок `Только временные` оставляет в списке только временные файлы Notepad++.

## Предпросмотр и полный просмотр

Наведение на элемент списка запускает быстрый предпросмотр. Если AI включен, приложение также может показать уже готовое summary или рассчитать его.

Клик по файлу открывает справа полную панель просмотра. В ней видно:

- имя файла;
- полный путь;
- время создания;
- время изменения;
- AI-тэги;
- AI-summary;
- содержимое файла.

Файлы внутри категорий открываются так же: нажмите на файл в карточке категории, и он появится справа.

## Временные и постоянные файлы

TempNoteManager различает:

- временные файлы Notepad++;
- постоянные файлы на диске;
- файлы, лежащие в backup-хранилище Notepad++.

Временный файл можно сделать постоянным:

1. Выберите временный файл.
2. Нажмите `Сохранить как...` в правой панели.
3. Выберите путь сохранения.
4. Приложение скопирует содержимое и обновит `session.xml`.

После этого Notepad++ будет считать файл постоянным.

Постоянный файл можно сделать временным:

1. Выберите постоянный файл.
2. Нажмите `Сделать временным`.
3. Подтвердите действие.
4. Приложение создаст временную копию в backup-папке Notepad++ и обновит `session.xml`.

## Экспорт временных файлов

Кнопка `Экспорт temp` копирует все доступные временные файлы в выбранную папку.

Это не меняет `session.xml` и не делает файлы постоянными. Это быстрый способ сохранить копии временных заметок перед разбором.

## Сортировка и порядок вкладок

Файлы можно перемещать:

- drag and drop в таблице;
- drag and drop в карточках;
- кнопками `↑` и `↓` для выбранного элемента.

После перемещения порядок меняется локально в TempNoteManager.

Чтобы записать новый порядок обратно в Notepad++:

1. Закройте Notepad++.
2. Нажмите `Порядок`.
3. Подтвердите запись в `session.xml`.

## Удаление файлов

Кнопка `Удалить` удаляет выбранный файл из списка Notepad++ и перемещает файл в корзину.

Режим корзины выбирается в верхней панели:

- `Windows` - системная корзина Windows.
- `Папка` - пользовательская папка-корзина.

Если выбран режим `Папка`, укажите путь рядом с полем `корзина` или нажмите `...`.

Перед удалением приложение показывает подтверждение с именем файла, путем и выбранной корзиной.

Если файл уже отсутствует на диске, TempNoteManager предложит удалить только запись из `session.xml`.

## Переименование файлов

Файл можно переименовать из:

- верхней кнопки `Переим.`;
- строки таблицы;
- карточки;
- файла внутри категории;
- правой панели просмотра.

При переименовании приложение:

- физически переименовывает файл на диске;
- обновляет путь в `session.xml`;
- сохраняет связь с AI-кэшем, если содержимое файла не изменилось.

## Категории-хранилища

Категории - это реальные директории на диске. Они нужны, чтобы быстро разобрать файлы по смысловым папкам.

Чтобы создать категорию:

1. Укажите корневую папку в поле `категории`.
2. Нажмите `+` в блоке `Категории`.
3. Введите название.
4. Выберите цвет или нажмите `Авто по описанию`.
5. Заполните `Описание для AI`.
6. Нажмите `Создать`.

Описание помогает AI понять, какие файлы относятся к категории.

Пример описания:

```text
Черновики SQL-запросов, миграции, выборки, заметки по структуре баз данных.
```

## Цвета категорий

Цвет можно выбрать вручную из списка или ввести в формате:

```text
#RRGGBB
```

Кнопка `Авто по описанию` подбирает цвет по тексту описания категории. Это локальная эвристика, отдельный AI-запрос для цвета не нужен.

## Перемещение файлов в категории

Чтобы переместить файл в категорию:

1. Перетащите файл из общего списка на карточку категории.
2. Подтвердите действие.

После подтверждения приложение:

- физически перемещает файл в директорию категории;
- обновляет путь в `session.xml`;
- обновляет список.

Чтобы вернуть файл из категории в общий список:

1. Перетащите файл из категории обратно в основной список.
2. Подтвердите действие.

Важно: при возврате из категории файл перемещается в общий корень категорий, а не в исходную папку, где он лежал до разбора.

## AI: подключение

AI-функции необязательны. Без AI приложение продолжает работать как менеджер файлов и сессии Notepad++.

Чтобы включить AI:

1. Поставьте флажок `AI`.
2. Укажите endpoint.
3. Укажите модель.
4. Введите API key.
5. Нажмите `Сохранить`.

Endpoint по умолчанию:

```text
https://api.openai.com/v1/chat/completions
```

Модель по умолчанию:

```text
gpt-4.1-mini
```

API key сохраняется в Windows Credential Manager:

```text
TempNoteManager.AI.ApiKey
```

В `settings.json` ключ не записывается.

Также поддерживаются переменные окружения:

```powershell
$env:TEMP_NOTE_AI_API_KEY = "..."
$env:OPENAI_API_KEY = "..."
$env:TEMP_NOTE_AI_ENDPOINT = "https://api.openai.com/v1/chat/completions"
$env:TEMP_NOTE_AI_MODEL = "gpt-4.1-mini"
```

Для локальных OpenAI-совместимых серверов можно указать свой endpoint, например:

```text
http://localhost:11434/v1/chat/completions
```

## AI-summary

Кнопка `AI summary` в правой панели пересчитывает краткое описание выбранного файла.

Если summary уже было рассчитано и файл не менялся, приложение берет результат из кэша.

При наведении на файл TempNoteManager также может показать summary во всплывающем предпросмотре.

## AI-тэги и классификация

Кнопка `AI тэги` анализирует видимые файлы и сопоставляет их с существующими категориями.

Важно: анализируются именно видимые элементы. Если включен поиск или `Только временные`, AI будет работать только с отфильтрованным списком.

Один файл может получить несколько тэгов и цветов, если AI считает, что он подходит к нескольким категориям.

Колонка `AI тэги` показывает найденные тэги. Эти тэги помогают быстро понять, куда файл, вероятно, относится.

## AI-подсказки категорий

Кнопка `AI категории` просит AI посмотреть на видимые файлы и предложить недостающие категории.

Сценарий работы:

1. Нажмите `AI категории`.
2. Посмотрите предложенные категории.
3. Отметьте нужные.
4. Нажмите `Создать выбранные`.

TempNoteManager создаст директории категорий в корневой папке категорий и сохранит их описания.

## AI-кэш и стоимость запросов

AI-кэш хранится здесь:

```text
%APPDATA%\TempNoteManager\ai-cache.json
```

Кэш привязан к SHA-256 содержимого файла. Поэтому перемещение или переименование файла не заставляет приложение заново анализировать summary и тэги, если содержимое не изменилось.

Категории версионируются по имени и описанию. Если создать новую категорию или изменить описание существующей, TempNoteManager переанализирует только новую или измененную часть классификации.

Практический совет: сначала создайте несколько понятных категорий с хорошими описаниями, затем запускайте `AI тэги`. Так будет меньше повторных запросов и точнее результат.

## Где хранятся данные

Настройки:

```text
%APPDATA%\TempNoteManager\settings.json
```

AI-кэш:

```text
%APPDATA%\TempNoteManager\ai-cache.json
```

AI API key:

```text
Windows Credential Manager -> Generic Credential -> TempNoteManager.AI.ApiKey
```

Сессия Notepad++:

```text
%APPDATA%\Notepad++\session.xml
```

Backup-папка Notepad++:

```text
%APPDATA%\Notepad++\backup
```

Перед записью изменений в `session.xml` приложение создает `.bak`-копию рядом с файлом.

## Типичный рабочий процесс разбора

1. Закройте Notepad++.
2. Откройте TempNoteManager.
3. Нажмите `Обновить`.
4. Включите `Только временные`, если сначала разбираете временные заметки.
5. Создайте несколько категорий с описаниями.
6. Нажмите `AI категории`, если хотите получить подсказки недостающих категорий.
7. Нажмите `AI тэги` для видимых файлов.
8. Просматривайте файлы справа и перетаскивайте их в категории.
9. Для важных временных файлов используйте `Сохранить как...`.
10. Для мусора используйте `Удалить`.
11. Если меняли порядок вкладок, нажмите `Порядок`.
12. Откройте Notepad++ и проверьте результат.

## Частые проблемы

### В списке нет файлов

Проверьте путь `session`. Если Notepad++ использует другой профиль или portable-версию, выберите `session.xml` вручную через кнопку `...`.

### AI выключен

Поставьте флажок `AI`, заполните endpoint, модель и API key, затем нажмите `Сохранить`.

### AI просит создать категории

Для `AI тэги` нужны категории с описаниями. Создайте хотя бы одну категорию в блоке `Категории`.

### Файл не перемещается в категорию

Проверьте, что файл существует на диске и что у приложения есть права записи в папку категории.

### Notepad++ не увидел изменения

Закройте Notepad++ перед записью изменений в `session.xml`. Если Notepad++ был открыт во время операции, он мог перезаписать сессию при закрытии.

### Кириллица в файле отображается странно

TempNoteManager читает текст для предпросмотра безопасным способом, но кодировка старых файлов может быть нестандартной. В таком случае проверьте файл в Notepad++ и при необходимости сохраните его в UTF-8.

## Безопасность

Не публикуйте в issue или скриншотах:

- AI API key;
- содержимое личных заметок;
- `settings.json`, если там есть приватные пути;
- `ai-cache.json`, если анализировались приватные файлы.

Для тестов с реальными заметками полезно сначала сделать копию профиля Notepad++ или работать с отдельной тестовой сессией.
