# Contributing

Спасибо за интерес к TempNoteManager.

## Локальная проверка

```powershell
dotnet restore
dotnet build .\TempNoteManager.csproj -c Release
```

## Рекомендации

- Перед изменениями логики `session.xml` сделайте резервную копию профиля Notepad++.
- Не коммитьте реальные AI-ключи, пользовательские настройки, логи и временные файлы.
- Сохраняйте поведение безопасным по умолчанию: операции удаления, перемещения и перезаписи должны быть подтверждаемыми.
- Для AI-функций учитывайте стоимость запросов и переиспользуйте кэш, когда файл не изменился.

## Pull request

В описании PR укажите:

- Что изменилось.
- Какие пользовательские сценарии проверены.
- Требуется ли миграция настроек или кэша.
- Есть ли риск для `session.xml` или физических файлов пользователя.
